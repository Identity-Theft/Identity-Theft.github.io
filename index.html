Mods

No XP Anvils

Removes the XP cost for repairing, renaming and applying enchantments to tools and gear in an anvil.

Infinity Rework

Reworks and balances the Infinity enchantment by splitting it into three tiers with each having a higher chance of saving an arrow.

Rail Destinations

Port of the RailSwitch plugin used on the CivMC server that allows rails to automatically switch directions based on a specified destination.

Create: Universal Ores

Create: Universal Ores is an addon for Create which adds new zinc ore variants for andesite, diorite, granite, tuff and calcite, in-line with Universal Ores.

Harvest Enchantment

Harvest Enchantment adds one new enchantment which allows hoes to harvest crops when right clicking.

Keep XP on Death

Keep your experience points on death. Updated from mastrocaleb's Keep XP on Death.

Bleached

Adds bleach which can be used to return wool, carpet, beds, banners and leather armour to their original colours.

Sculk Items

Sculk Items was planned to add more to the Ancient Cities but is no longer in development.
